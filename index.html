RK Works Richkid Racingparts and Tuning POS and Inventory System

The system streamlines sales transactions, inventory tracking, supplier and customer management, service records, and report generation. 03

AWBS

AWBS (Apartment Water Billing System) is a desktop-based billing and customer management system designed to automate the water billing operations of a water service provider. The system enables administrators to manage customer records, generate and monitor water bills, process payments, maintain user accounts, and generate reports. It provides real-time dashboard analytics, including customer statistics, paid and pending bills, and total collections, helping improve billing accuracy, operational efficiency, and record management.

C# MS SQL Server
04

Aklatura

Aklatura is a Library Management System created to help school libraries run more efficiently. Designed to improve how library resources are organized, accessed, and monitored, it serves as a centralized platform for students, professors, and librarians that provides easy access to knowledge and acts as an efficient partner in managing library resources. Simplifying borrowing and returning process, ensures timely notifications for overdue items, and maintains accurate records of transactions. Aklatura also supports informed decision-making through its report and analytics features, helping librarians analyze trends, manage inventory, and allocate resources wisely. With its user-friendly interface and focus on accessibility, Aklatura enhances that overall learning experience and promotes a culture of responsible library use.

ASP.NET MS SQL Server

NutriBuddy

NutriBuddy is a mobile fitness and nutrition application designed to help users achieve healthier lifestyles through personalized meal planning and fitness tracking. The application provides secure user authentication, health profile assessment, personalized meal recommendations, calorie monitoring, and a nutrition calendar. By utilizing user information such as age, gender, height, weight, and fitness goals, NutriBuddy delivers customized dietary guidance that encourages healthier eating habits and long-term wellness.

Figma

BrightBuds

BrightBuds is a mobile application designed to support children aged 7–14 diagnosed with predominantly inattentive ADHD by promoting routine-building, emotional regulation, and behavioral consistency. The system integrates therapist-guided task management, mood journaling, and gamification through a reward-based aquarium progression system, enabling therapists and parents to monitor the child's progress while extending therapeutic support into the home environment.

Flutter Dart Firebase
